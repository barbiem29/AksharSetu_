# AksharSetu_

"अक्षरं ब्रह्म परमं — The eternal syllable is the highest reality."

AksharSetu is a web application that bridges traditional Hindi learning with modern digital interaction. It is built to connect script, sound, and meaning into a unified learning experience, making Hindi accessible through intuitive and interactive design.

The platform combines cultural depth with modern frontend technologies, enabling users to explore the Hindi language through visual, auditory, and text-based interactions.

Features:

* Alphabet Learning: Complete Hindi varnamala with pronunciation support using browser-based text-to-speech.
* OCR and Audio: Extract Hindi text from uploaded images and convert it into speech.
* Text Input Mode: Type Hindi text and instantly listen to accurate pronunciation.
* Translation Support: Basic word-level English translation for better understanding.
* Interactive Interface: Smooth UI with responsive layouts and animated feedback.

Technology Stack:

* React.js (component-based UI architecture)
* HTML5, CSS3 (UI structure and styling)
* JavaScript (application logic)
* Web Speech API (text-to-speech functionality)
* Tesseract.js (Hindi OCR for image-to-text extraction)

How to Run:

For React Version:

* npm install
* npm start

For Standalone Version:

* Open the HTML file directly in a browser

Project Purpose:

AksharSetu demonstrates how traditional language learning can be enhanced using modern web technologies. It focuses on simplicity, usability, and meaningful interaction rather than unnecessary complexity.

Future Improvements:

* Higher accuracy OCR using advanced APIs
* Full sentence-level translation
* Enhanced UI/UX with improved accessibility
* Voice input support for Hindi speech recognition

This project reflects an effort to digitally preserve and interact with Hindi through a seamless blend of tradition and technology.
